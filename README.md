tpts-joark-mottak
================

Tar i mot tynne kafka meldinger fra Joark og produserer tykke kafka meldinger for videre behandling.

---

# Komme i gang

Gradle brukes som byggverktøy og er bundlet inn.

`./gradlew build`

---
